## 📚 Main Project: LibraLens
**LibraLens** is an LMS (Library Management System) designed to support librarians in managing books, borrowings, and returns.  
Key features include:
- 🧠 AI recommendation engine using `sentence-transformers` and `cosine similarity`
- 📈 Suggests new books not in the library based on user borrowing patterns
- 📧 Email reminders for overdue books
- 🔒 Secure multi-client-server communication
- 🟢 Status indicators for borrowed/returned books
