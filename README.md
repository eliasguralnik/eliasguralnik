### 📦 Guralnik Mailer — Open Source NPM Package
> *Node.js · MedusaJS v2 · Multi-Provider · Zero-Config*

A **premium transactional email engine** for MedusaJS v2 and Node.js ecosystems, published on NPM. It solves one of the most underestimated pain points in e-commerce: reliable, beautiful, multi-language transactional emails — with zero boilerplate.

**Key capabilities:**
- ✉️ **10 polished themes** out of the box
- 🌐 **10 languages** supported with automatic locale extraction
- 🔀 **Multi-provider routing** — Resend, SendGrid, SMTP — switchable without code changes
- ⚙️ Drop-in integration: configure once, it just works

This is infrastructure-level thinking applied to developer experience.

### 🔭 LibraLens — AI-Enhanced Library Management System *(In Development)*
> *Python · Sockets · SSL/TLS · Machine Learning · Multi-Client Architecture*

LibraLens is where systems programming meets applied AI. It's a multi-client library management platform built on raw Python sockets with SSL/TLS encryption — no web framework abstractions, just the protocol stack.

**What makes it technically interesting:**
- 🤖 **AI Recommendation Engine** powered by `scikit-learn` and `sentence-transformers` using **cosine similarity** on semantic embeddings — not keyword matching
- 🔒 **Secure socket communication** with full SSL/TLS handshake implementation
- 📡 **Real-time status monitoring** across a multi-client architecture
- 🧩 Designed to demonstrate what happens when you understand the network layer and apply it deliberately

This is the kind of project that only makes sense if you've studied how networks actually work.
